# catdogs.club
## 简介：
1. 针对猫狗等小动物的社区论坛、资讯网站
2. 资讯：原创 + 转载
3. 论坛/问答社区
4. 后期：商城、医生问答
5. 盈利模式：广告联盟 销售抽成
## 开发：
前端：react ts antd
后端：go py node
## UI设计：
字体：[思源](https://github.com/adobe-fonts/source-han-sans)，因中文字体文件过大，建议不要大篇幅使用动态字体，仅在标题、菜单等一些地方使用即可，需注册[typekit](https://www.typekit.com/)使用。
